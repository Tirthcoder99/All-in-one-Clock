# All-in-One Clock

A fully functional clock website featuring:  

- **Year Countdown** – Shows how much time is left in the ongoing year with a New Year celebration animation.  
- **Timer** – User-set countdown timer with start, stop, and reset.  
- **Stopwatch** – Start, stop, and reset stopwatch with milliseconds precision.  
- **World Clock** – Displays current time in multiple cities around the world (New York, London, Tokyo, India).

## Demo

Once deployed on GitHub Pages or Netlify, you can access the live site.

## Features

1. **Year Countdown**
   - Automatically updates every second.
   - Triggers celebration when the year ends.
2. **Timer**
   - User can set minutes and seconds.
   - Alerts when timer finishes.
3. **Stopwatch**
   - Counts up in minutes, seconds, and milliseconds.
   - Can pause and reset anytime.
4. **World Clock**
   - Displays accurate time for multiple time zones.
